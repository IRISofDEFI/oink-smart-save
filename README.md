# OinkAI 🐷

AI-powered savings on Arc. Lock USDC, save smarter, talk to your money.

## What it is

OinkAI is a chat-based AI agent that helps users lock USDC on the Arc blockchain for a chosen duration. Like a digital piggy bank — but smart. You tell it what you want to save and how long, and it handles the rest.

## Why it exists

Saving is hard. Saving crypto is harder — too easy to move, too tempting to touch. OinkAI uses a time-locked smart contract on Arc so users can commit to saving, knowing the funds won't be available until the lock expires.

## Built on

- **Arc Network** — Stablecoin-native Layer 1 with deterministic finality and USDC as native gas
- **Circle Developer-Controlled Wallets** — Secure wallet infrastructure
- **OpenAI/Anthropic** — AI agent for natural language interaction (TBD)
- **App Kits** — Circle's SDK for USDC transfers
- **Lovable + Claude Code** — Solo-dev AI-assisted build workflow
